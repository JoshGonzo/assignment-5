How to Apply CSS to HTML

You can apply an external CSS file to HTML using <link> 

 <link rel="stylesheet" href="home.css">

How to Use JavaScript in a Webpage

You can use an external JavaScript file by using <script> 

<script src="home.js"></script>
Using the same tag you can also write JavaScript in html such as 
    <script>
        fwindow.addEventListener('scroll', () =>{
            if(scrollY >= 180){
                navbar.classList.add('bg')
            }else{
                navbar.classList.remove('bg')
            }
            console.log(scrollY);
        })
    </script>
